About Air Quality



HTML5 and CSS

We are using HTML5 and CSS to present the design of the website. We use a lot of features that are included in the Boostrap CSS framework. Learn more about Boostrap by clicking here.

JavaScript Framework

For our JavaScript framwork we opted into using vue.js. Vue is an open-source JavaScript framework. Learn more about vue.js by click here.

API

The API we used came from The World Air Quality Index project. AQICN is a non profit organization that focuses on providing reliable data for air quality measurements around the world. They provided us with a free to use API. To learn more about their API click here.
